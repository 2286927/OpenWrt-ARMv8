# Telegram：https://t.me/OpenWrt_Yuan
## 版本更新github上传固件，插件更新Telegram频道上传
# 默认IP：10.10.10.10
## 写入EMMC:
**cd /root**

**./install-to-emmc**

## 升级：
**把 update-amlogic-openwrt.sh 及 img镜像上传至 /mnt/mmcblk2p4**

**cd /mnt/mmcblk2p4**

**chmod 755 update-amlogic-openwrt.sh**

**./update-amlogic-openwrt.sh xxxxxx.img**
